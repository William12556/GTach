# Workspace

Execution space for governance framework documents.

## Structure

| Folder | Purpose |
|--------|---------|
| admin/ | Administrative reports (excluded from git) |
| requirements/ | Requirements documents |
| design/ | Design documents (Tier 1-3) |
| change/ | Change documents |
| knowledge/ | Institutional knowledge base |
| issues/ | Issue documents |
| proposal/ | Proposals (excluded from git) |
| prompt/ | T04 prompt documents for Claude Code |
| trace/ | Traceability matrix |
| audit/ | Audit reports |
| test/ | Test documentation and results |
| ai/ | Optional AI workspace (excluded from git) |

## Document Lifecycle

- Active documents reside in `<class>/`
- Closed documents move to `<class>/closed/`
